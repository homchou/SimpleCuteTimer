# SimpleCuteTimer

![Timer](https://raw.githubusercontent.com/homchou/SimpleCuteTimer/master/D2.ico)  
* 简单魔方计时器 .net winform实现（之前在学校练习版本）
* 自动生成打乱公式(25位)
* 支持计时器全屏，方便比赛使用
* 绿色免安装（因已开源不会有不安全，如打开有安全提示请忽略）
* Microsoft .NET Framework 2.0，在 Windows下Win7或以上都可使用
